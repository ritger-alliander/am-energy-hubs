Dataproducten (Product) ()
Dataproducten worden beschikbaar gesteld via waardestroom "7) Beschikbaar stellen van netbeheerdata". De volgende informatie moet beschikbaar zijn als onderdeel van het solution design: * Naam van het dataproduct * Dataset beschreven en gepubliceerd op netbeheer-nederland.github.io (link opgenomen) * Dataservice verantwoordelijk voor het leveren van de datasets (centraal of decentraal) * Voorwaarden waaronder de dataset wordt geleverd (open/gesloten data)
